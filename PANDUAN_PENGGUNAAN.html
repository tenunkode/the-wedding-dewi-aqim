📋 Manajemen Daftar Tamu

Halaman ini digunakan untuk menyimpan daftar tamu secara permanen di browser Anda dan mengirimkan pesan undangan yang sudah terpersonalisasi.

Link Akses: list-nama.html
1 Menambah Tamu Baru

Klik tombol (+) Merah Muda di pojok kanan bawah. Masukkan nama lengkap tamu, lalu tekan Simpan.

  • Data akan tersimpan secara otomatis di memori browser Anda (Local Storage).
  • Nama yang sudah ada tidak perlu dimasukkan lagi di masa mendatang.
2 Mencari & Memfilter

Gunakan kolom pencarian di bagian atas untuk menemukan nama tamu dengan cepat.

  • Jika nama tidak ditemukan di daftar, sistem akan membuatkan slot "MANUAL" agar Anda tetap bisa mengirim undangan tanpa menyimpannya ke daftar permanen.
3 Mengirim Undangan

Setiap nama tamu memiliki dua tombol utama:

  • Salin : Menyalin teks undangan ke clipboard (untuk di-paste manual).
  • WA : Membuka aplikasi WhatsApp dan langsung mengisi pesan untuk tamu tersebut.

☁️ Sinkronisasi Cloud (Firebase)

Sistem ini sudah terhubung secara otomatis dengan akun Firebase yang sama dengan fitur Kirim Ucapan.

1 Otomatis Aktif

Anda tidak perlu melakukan pengaturan kunci API lagi. Sistem sudah mengambil data dari index.html.

2 Keamanan Data

Daftar tamu disimpan dalam koleksi database yang aman. Hanya orang yang memiliki akses ke link manajemen ini yang bisa mengedit daftar tamu.

3 Real-time Sync

Setiap perubahan (tambah/hapus tamu) akan langsung tersimpan di Cloud dan muncul di HP lain yang membuka halaman ini tanpa perlu refresh.

💡 Tips Penting

  • Sinkronisasi Real-time: Dengan Firebase, daftar tamu akan otomatis sinkron antar HP tanpa perlu refresh.
  • Support GitHub Pages: Sistem ini sekarang 100% kompatibel dengan GitHub Pages.
  • Peringatan Keamanan: Jangan lupa ubah Rules di Realtime Database Anda ke mode produksi jika acara sudah selesai agar data tidak bisa diakses orang lain.
  • Cetak PDF: Tekan Ctrl + P untuk menyimpan panduan ini sebagai PDF.
